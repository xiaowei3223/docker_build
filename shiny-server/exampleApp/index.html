Welcome to shiny-server built by Mora-Lab

This shiny-server includes two parts.

  • 1. Shiny-server: to run one or more of your shinyApp automatically.
  • 2. RStudio-server: to edit the shinyApps or install R packages for running the shinyApps.

As we designed, you can use account `rstudio` with password rstudio to log in the RStudio-server.

If you want add your shinyApp, you just put your shinyApp to `/home/rstudio/shinyApps` folder in the container.

After installed R packages for the shinyApp with rstudio account, you can view your shinyApp on the website http://host_ip:3838/your_shinyApp.
